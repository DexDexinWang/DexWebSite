A constant(final in Java) variable could not be changed after a value is assigned to this constant variable. Also, it should not be changed during program processing.

Operators

Arithmetic operators: +, -, *, /, %, ++, –,

Relation operators: ==, !=, >, <, >=, <=

Logic operators: &&, ||, !

Bit operators: &, |, ^, ~

E.g.

A = 0011 1100

B = 0000 1101

A&B = 0000 1100

A|B = 0011 1101

A^B = 0011 0001

~A = 1100 0011

A<<2 = 1111 0000

A>>2 = 00001 111

Assignment Operators: =, +=, -=, *=, /=, %=, <<=, >>=, &=, ^=, !=
