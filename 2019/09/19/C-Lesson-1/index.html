Setup CSC Command

CSC command is used to compile .cs files.

E.g.

>csc helloword.cs

If the Command Prompt shows {‘csc’ is not recognized as an internal or external command}, you have to setup Environment Variable with “Path”.

>C:\Windows\Microsoft.NET\Framework{version}\

Identifier

Identifiers are used to identify classes, variables, functions, and other user defined projects. It should be following below rules:

  • Identifier starts at letter, underline, or @ sign, following by letters, numbers, underline, or @
  • The first character should not be a number
  • It should not contains any space and other special characters such as ?, +, #, %,^.
  • Indentifier should be different from Keywords
  • Indentifier is case sensitive
  • Indentifier should not be the same as libary class names.

Object Type

Object type is the lowest basic class in Common Type System. Object is alias of System.Object. The Object variable could be used to save all other classes.
However, when a value is asigned to an object variable, it should do Boxing. Also, when an object variable is used transfered as another object, we call it Unboxing.

Dynamic Type

Dynamic Type also supplies the similar method to define a variable that could save a value in all types.

Object Type will be checked when the program is compling. Dynamic Type will be checked when the program is running.

String Type

In C#, users could assign value to string in 2 ways such as “” or @””.

Pointer Type
