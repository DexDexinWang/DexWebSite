Stack

Stack is a coolection or structure to hold multiple elements prior to LIFO processing.

  • Push() - push at the top
  • Pop() - pop at the top
  • Peek() - look at the top without poping it out

Stack is defined as a legacy class because it has performance problem by extendeing Victor class. It is replaced as Deque Interface which extends Queue Interface.
